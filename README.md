# Куток

Пошук співмешканців і вільних кімнат у Києві — без ризику й зайвого тертя.

- **Для кого:** 22–30, переїзд у Київ або вже тут; головне — довіра.
- **Модель:** двостороння — оголошення кімнат + профілі шукачів.
- **Довіра:** верифікація телефону й соцмереж, відгуки після заселення, бейджі.
- **Комунікація:** вбудований чат, контакти — лише за взаємною згодою.
- **Платформа:** мобільний веб (mobile-first), далі адаптив до десктопу.
- **Тон:** теплий і домашній — «знайти свій куток».
- **Статус:** етап документів (бриф → дослідження → далі карта екранів і флоу).

## Дослідження — коротко

- **Конкуренти (3 групи):** ХАРД — OLX.ua, DIM.RIA, HomeMate, Telegram-канали сусідів, Facebook-групи; СОФТ — Flatfy/LUN, Bird; АСПІРАЦІЙНІ — SpareRoom, Badi, Roomi, Diggz, Roomster.
- **Метод:** по кожному — окремий веб-ресерч за 5 осями (аудиторія, продукт, механізм, довіра, монетизація) + незалежна перевірка джерел. 49 із 60 осей підтверджено, решта позначені «дані не підтверджені».
- **Головний висновок:** ніша «довіра до людини-співмешканця + двостороння модель + фільтри за звичками» на київському ринку не зайнята — усі або каталог обʼєктів (OLX, DIM.RIA), або стрічка без довіри (Telegram, Facebook), або далекі міжнародні гравці.
